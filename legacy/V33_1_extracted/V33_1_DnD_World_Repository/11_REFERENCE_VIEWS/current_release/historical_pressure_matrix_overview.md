# Historical Pressure Matrix Overview

V33_1 keeps the world’s time-depth framework visible as a GM-facing reference surface.

## Core idea
The present is where inherited history and future possibility meet.

## Two axes
### Scale
Person -> Relationship -> Community -> Institution -> State -> Globe

### Time
Event -> Memory -> Record -> Inheritance -> Reaction -> Myth -> Ruin -> Revival

## What this framework is for
Use it whenever you need to explain why something exists now:
- a law
- a feud
- a ruined district
- a faction obsession
- a ritual taboo
- a noble rivalry
- a route danger
- a local custom that no one fully remembers

## Three tempos
- **Immediate Time:** present-moving consequence
- **Accumulated Time:** repeated action becoming pattern
- **Inherited Time:** people beginning inside conditions they did not create

## GM shortcut
For any current problem, ask:
1. What event started pressure?
2. Who remembers it and how differently?
3. What institution recorded or distorted it?
4. What did descendants inherit?
5. What reaction, backlash, or reform did it create?
6. What part has now hardened into myth, ruin, or revival?
