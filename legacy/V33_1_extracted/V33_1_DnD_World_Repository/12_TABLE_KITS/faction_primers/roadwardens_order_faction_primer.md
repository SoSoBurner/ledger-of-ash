# Roadwardens Order Faction Primer

**Identity:** Roadwardens Order

**What it wants:** Keep roads safe, commerce moving, and Shelk’s authority visibly credible.

**How it operates:** route patrols, checkpoint presence, escort work, barracks deployment, public security at noble events

**What it fears:** Their visibility creates order, but also makes them the first target for anyone trying to embarrass House Shelk.

**What it is doing now:** caravan disruption, celestial enforcer rumors, banditry, need to police without choking trade

**How it reacts to party interference:** If the party helps, it will try to formalize that help into obligation. If the party obstructs it, it will answer through the same methods it uses to preserve leverage.

**Best first conflict:** A routine request becomes factional the moment the party changes pace, access, or witness control in a space this faction treats as governed.
