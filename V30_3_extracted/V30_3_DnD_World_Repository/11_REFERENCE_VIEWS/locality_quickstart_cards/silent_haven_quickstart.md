# Silent Haven quickstart card

**Polity:** Principality of Roaz
**Visual read:** Fortified coastal settlement shaped by watchfulness and maritime necessity.
**Behavioral read:** keep quiet order and comply with security
**Danger read:** sea threat; northern route vulnerability
**Who matters first:** Roazian enforcers
**Easiest playable complication:** A minor official wants paperwork clarified.
**Best opener:** A first arrival at Silent Haven immediately highlights remote roazian coastal refuge and naval defense node where antimagic caution and quiet discipline are palpable
