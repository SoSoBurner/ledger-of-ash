# Silent Haven

**Polity:** Principality of Roaz

**Arrival:**** Fortified coastal settlement shaped by watchfulness and maritime necessity.

**Inside:**** Refuge logic, dock vigilance, and law-heavy calm define it.

**Economic role:** naval defense, route security

**Law and enforcement feel:** strict Roazian order

**Religious and cultural texture:** subdued

**Active tensions:** sea threat, northern route vulnerability

**Named NPC slice:** Helm Steelmarch, Talia Stonewake

**Linked routes:** Ithtananalor - Silent Haven convoy lane

**World-bridge block:** Silent Haven matters because it anchors Principality of Roaz in the wider world, carries pressure beyond its own streets, and can redirect trade, law, rumor, and campaign movement if destabilized.
