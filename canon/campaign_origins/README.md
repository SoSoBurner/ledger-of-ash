# Ledger of Ash background origin openers — in-world prose

These files rewrite each background opener as game-ready in-world text.

Design constraints preserved:
- Each background begins in its repo-mapped origin locality.
- The opener stays rooted in the locality and only uses adjacent localities as corroboration or route pressure.
- Locality and polity are not flattened into each other.
- Rosalind is directly staged only for Shelkopolis origins.
- Outside Shelkopolis, Rosalind appears only as distant rumor carried by travelers, with reverence, lust, admiration, or envy.
- Every opener leads naturally toward the Stage II Fairhaven transition.
