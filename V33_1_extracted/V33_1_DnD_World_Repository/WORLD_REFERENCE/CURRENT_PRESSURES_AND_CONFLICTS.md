# Current Pressures and Conflicts

- magical convenience vs magical contamination in the Sheresh north
- quota stability vs labor resentment inside Soreheim infrastructure
- custom, marriage law, and prestige exchange on Phoenix Island
- contract memory and reputation warfare along Nomdara routes
