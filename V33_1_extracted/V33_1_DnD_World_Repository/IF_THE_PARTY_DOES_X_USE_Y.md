# If the Party Does X, Use Y

- If the party wants the fastest possible opener, use `RUN_A_SESSION_NOW.md`.
- If the party enters a city and asks basic questions, use the relevant file in `12_TABLE_KITS/arrival_kits/`.
- If the party ignores procedure or offends authority, use the matching file in `12_TABLE_KITS/escalation_ladders/`.
- If the party travels, use the matching route card in `11_REFERENCE_VIEWS/current_release/route_runner_cards/` and the matching file in `12_TABLE_KITS/travel_complications/`.
- If the party follows rumor, use the relevant file in `12_TABLE_KITS/rumor_starters/` and then jump to a linked starter packet.
- If the party wants a clearer GM card, use `PLAY/TABLE_READY_LOCALITY_CARDS/`.
- If the party changes a place permanently, record it in `CURRENT_CAMPAIGN_STATE.md`, `CHANGED_LOCALITIES.md`, and `UNRESOLVED_PRESSURES.md`.
