# V30_8 Release Notes

## Release identity
V30_8 is a **launch-surface and continuity-deepening release**. It does not treat the package’s next need as broad new lore. It treats the repository as already broadly playable at the world layer and focuses on the safer, higher-value work of converting asserted canon into stronger table surfaces.

## What changed
- Added starter packets for 16 already surfaced localities whose support stack already existed in the package.
- Filled the root campaign persistence files with active runtime content for the House Shelk vs the Union campaign.
- Added a current-campaign packet set for the live Shelkopolis arc.
- Added missing high-value faction primers where table utility lagged behind asserted faction presence.
- Added a production-ready priority analysis that formalizes the next canon-safe content lane.

## Release doctrine
This release favors:
- play usefulness over ornamental expansion
- continuity stability over speculative breadth
- retrieval reliability over lore sprawl
- canon-safe deepenings of asserted structures over unsupported invention
