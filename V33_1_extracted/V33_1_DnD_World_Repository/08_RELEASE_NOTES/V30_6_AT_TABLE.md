# V32_4 At Table

## What got easier to run
- There is now one obvious GM entrypoint.
- The browse path starts with operator-facing files instead of governance files.
- Fast-start kits now exist for arrivals, travel pressure, rumors, and escalation.

## What browse path changed
Start with `START_HERE_GM.md`, then move through world overview, settlement ledger, polity dossiers, locality quickstarts, route runner cards, session walkthroughs, and table kits.

## What duplicates were removed
The active duplicate browse artifact for the Shelkopolis / Plume’s End northern road variant was removed in favor of a single canonical card.

## What table kits were added
- polity primers
- faction primers
- arrival kits
- travel complications
- rumor starters
- escalation ladders

## What still should not be treated as fully mature
Deep governance files remain essential for audit and system maintenance, but a new GM should not need them to start play.
