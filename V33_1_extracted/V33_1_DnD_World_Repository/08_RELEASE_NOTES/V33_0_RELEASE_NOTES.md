# V33_1 Release Notes

## Summary
V33_1 is a corrective hardening and canon-deepening release built from the V32_8 repository package. It keeps the repository standalone while repairing current-release identity, improving semantic integrity in NPC records, and strengthening canon translation into usable GM-facing reference surfaces.

## Major changes
- normalized current package identity to `V33_1` / `V33_1_DnD_World_Repository`
- regenerated current-release manifest and validation surfaces
- repaired anonymous interface-role placeholders into intentionally generic locality roles
- corrected a cluster of named NPC records whose role fields incorrectly held other NPC names
- added focused reference files for pantheon, religion, doomsday consequences, and regional play
