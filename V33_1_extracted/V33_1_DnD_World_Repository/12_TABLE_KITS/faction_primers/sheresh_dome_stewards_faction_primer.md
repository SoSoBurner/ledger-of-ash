# Sheresh Dome Stewards Faction Primer

**Identity:** Sheresh Dome Stewards

**What it wants:** Keep domes alive first, comfortable second, and politically stable throughout crisis.

**How it operates:** maintenance rotations, containment enforcement, resource triage, commune administration

**What it fears:** Their authority is strongest in emergencies, but repeated emergencies make every order feel personal.

**What it is doing now:** dome strain, mutation anxiety, repair fatigue, public panic after losses

**How it reacts to party interference:** If the party helps, it will try to formalize that help into obligation. If the party obstructs it, it will answer through the same methods it uses to preserve leverage.

**Best first conflict:** A routine request becomes factional the moment the party changes pace, access, or witness control in a space this faction treats as governed.
