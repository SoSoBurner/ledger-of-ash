# Engineers Consortium Faction Primer

**Identity:** Engineers’ Consortium

**What it wants:** Keep the Alliance physically functional and expansion-capable through disciplined engineering.

**How it operates:** surveying, tower maintenance, span construction, machine design, repair scheduling

**What it fears:** Public trust depends on their competence, so visible failure quickly becomes political and spiritual embarrassment.

**What it is doing now:** resource strain, sabotage risk, weather damage, need to maintain expansion without catastrophic failure

**How it reacts to party interference:** If the party helps, it will try to formalize that help into obligation. If the party obstructs it, it will answer through the same methods it uses to preserve leverage.

**Best first conflict:** A routine request becomes factional the moment the party changes pace, access, or witness control in a space this faction treats as governed.
