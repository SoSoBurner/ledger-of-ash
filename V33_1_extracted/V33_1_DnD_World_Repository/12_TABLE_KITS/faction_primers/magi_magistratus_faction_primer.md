# Magi Magistratus Faction Primer

**Identity:** Magi Magistratus

**What it wants:** Keep dangerous or illicit magic within legible boundaries and preserve Shirsh authority over magical investigations.

**How it operates:** inquiry, licensed detection, case review, field investigation

**What it fears:** The deeper the investigation goes, the more the faction risks becoming a political weapon instead of a neutral safeguard.

**What it is doing now:** hidden practitioners, house politics, information control

**How it reacts to party interference:** If the party helps, it will try to formalize that help into obligation. If the party obstructs it, it will answer through the same methods it uses to preserve leverage.

**Best first conflict:** A routine request becomes factional the moment the party changes pace, access, or witness control in a space this faction treats as governed.
