# V33_1 Psycho-Social Chronology System Guide

## What changed
This release adds a non-destructive chronology layer that reads named NPCs as living entities shaped by routine, class, money, authority, memory, fear, desire, and repeated compromise.

## Where to look
- `05_RUNTIME_ENGINE/psycho_social_chronology/npc_living_entity_profiles.jsonl`

## GM operating loop
1. Start with the baseline NPC dossier.
2. Open the living-entity profile for the same NPC.
3. Use fear, desire, status, money, and routine pressure to decide how they react.
4. Push the outcome one scale upward only if the actor has enough reach.
5. Surface the result as custom, paperwork, price, force posture, rumor, or institutional habit.

## Safety rule
These files explain modern canon; they do not silently revise it.
