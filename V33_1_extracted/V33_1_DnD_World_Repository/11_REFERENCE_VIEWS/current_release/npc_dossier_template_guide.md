# NPC Dossier Template Guide

V33_1 internalizes the uploaded NPC template as a package-native authoring contract.

## Design rule
When useful, keep two layers:
- **Canon / public layer** for what is openly known, documented, or believed
- **GM / hidden layer** for secrets, private motives, concealed loyalties, and campaign-facing truth

## Minimum dossier sections
1. Core identity
2. Setting placement
3. Public face versus private truth
4. Psychology
5. Morality in practice
6. Resources and leverage
7. Relationships and rivals
8. Table-use hooks
