Build the framework around this core premise:

The world is shaped by two forces acting at once:
Scale determines where influence moves.
Time determines how influence survives.

Scale moves across living layers:
Person -> Relationship -> Community -> Institution -> State -> Globe

Time moves across historical persistence:
Event -> Memory -> Record -> Inheritance -> Reaction -> Myth -> Ruin -> Revival

The setting should feel deep by showing that every present condition is the meeting point between:
- current action
- stored consequence
- inherited pressure
- future reinterpretation

The central philosophical statement to preserve is:
The present person is where inherited history and future possibility meet.

# IX. Character Integration Rule

State that no important character should exist outside historical pressure.

For any meaningful character, define:
- what they personally want
- what relationship binds or wounds them
- what community shaped them
- what institution classified or trained them
- what state burden they live under
- what global condition frames their life
- what inherited history they carry without choosing

Preserve this formula:
Present desire + inherited pressure + available myth = dramatic identity

# X. Faction Integration Rule

State that every faction should exist in all five historical layers:
- Immediate Time
- Living Memory

- Inherited Structure
- Mythic Time

# XI. Location Integration Rule

State that every major place should be built across scale and time.

For each city, district, ruin, temple, archive, borderland, or trade route, define:
- founding event
- remembered trauma or pride
- official record
- inherited custom or law
- living stigma or privilege
- myth attached to the place
- present conflict
- future pressure building beneath the surface

Preserve this formula:
Place = event sediment + social memory + enforced structure + active pressure

# XII. Law and Institution Rule

State that any law that matters should have a history.

For every meaningful law, ask:
- What fear created it?
- What event justified it?
- Who recorded it?
- Who benefited?
- Who was categorized by it?
- What backlash did it produce?
- Does anyone still believe the original reason?

Note that this is especially useful for magic systems, caste systems, border control, religion, inheritance law, and trade regulation.

# XIII. Narrative Use

Explain how to make a conflict feel deep by showing it at multiple layers at once.

A strong conflict should not only be a present disagreement. It should also be:
- a relational fracture
- a local custom
- an institutional precedent
- a state interest
- a broader world pressure
- a historical inheritance
- a mythic justification

Explain that this is how the world feels alive.

Condense the model into:
- The title: The Historical Pressure Matrix

- Principle: History is prior matrix activity that did not disappear.
- Character truth: A person is both a source of new pressure and a recipient of old pressure.
- Narrative truth: The present is where inherited history and future possibility meet.
