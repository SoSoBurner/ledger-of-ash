# Batch 25 Progress

## What was improved

This batch focuses on Stage I and Stage II player experience rather than only route-scope claims.

### Settlement content
Each active rooted locality now has six explicit POI/service categories:
- recovery
- equipment
- information
- progression
- flavor
- secret

These are surfaced in the Services panel and tied to live actions.

### Itemization and readiness
Runs now track:
- inventory
- equipment
- class-specific readiness signals

This improves class fantasy, readability, and early/mid-early decision texture.

### Character embodiment
Character creation now sets:
- age
- presentation
- starting loadout
- life overview

That overview is shown in the Sheet so the run starts from a more embodied and world-rooted place.

## Current known limitations

- the rooted locality count is still 10, not full world coverage
- named NPC placement is still partial
- Stage III–V remain structurally present but not fully authored
- deterministic authority is still browser-local
