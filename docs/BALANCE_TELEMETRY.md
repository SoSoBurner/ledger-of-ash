
# Balance Telemetry

Tracked in this overlay:
- stage entry day
- stage turns
- stage actions
- stage travels
- stage route scouts
- stage safe-zone leverage uses
- stage encounters
- stage encounter wins
- stage retreats
- stage rescues
- stage creature wins
- stage hazard wins
- total levels gained
- total legend runs

Telemetry is currently stored in save state and summarized in the Character sheet and Hall of Legends.
