The current release uses the uploaded locality matrix build spec as the canonical locality contract. Start with:

1. `02_CANON_BASELINE/locality_matrix_backbone/locality_matrix.json`
2. `02_CANON_BASELINE/locality_matrix_backbone/route_matrix.json`
3. `02_CANON_BASELINE/locality_matrix_backbone/label_crosswalk.json`
4. `02_CANON_BASELINE/locality_matrix_backbone/validation_report.json`
