# Gods, Religions, and Divine Pressure

Religion in this repository is social infrastructure, emotional control, memory management, prophecy, ambition, compassion, indulgence, and fear.

## GM angle
Use religion not as color only, but as a way institutions justify sacrifice, hope, and control.

- local rites
- what this polity believes Doomsday meant
