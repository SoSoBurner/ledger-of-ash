# Red Hood Guild Faction Primer

**Identity:** Red Hood Guild

**What it wants:** Exploit the gaps created by rigid law, secret demand, and elite hypocrisy.

**How it operates:** smuggling, covert access, information exchange, deniable operations

**What it fears:** The guild thrives on opacity; every exposed cell threatens the myth of reach that protects the whole network.

**What it is doing now:** law crackdowns, competition with official power, need to stay deniable

**How it reacts to party interference:** If the party helps, it will try to formalize that help into obligation. If the party obstructs it, it will answer through the same methods it uses to preserve leverage.

**Best first conflict:** A routine request becomes factional the moment the party changes pace, access, or witness control in a space this faction treats as governed.
